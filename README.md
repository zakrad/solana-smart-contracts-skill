# solana-smart-contracts

[![skills.sh](https://skills.sh/b/zakrad/solana-smart-contracts-skill)](https://skills.sh/zakrad/solana-smart-contracts-skill)

A Claude Code skill for Solana smart contract development, generated from the [RareSkills 60-Day Solana Book](https://rareskills.io/solana-tutorial) (65 modules).

Covers installation, Rust fundamentals, account model, PDAs, SPL tokens, Token-2022, CPIs, security checks, native Rust, Borsh, sBPF internals, and more — with complete code examples.

## Install

```bash
npx skills add zakrad/solana-smart-contracts-skill
```

## Usage

In any Claude Code session:

```
/solana-smart-contracts
```
